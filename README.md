## ✨ 游戏特色

| 功能 | 描述 |
|------|------|
| 🎨 **马卡龙色系** | 粉色、蓝色、黄色、紫色、绿色，超可爱画风 |
| 🎯 **10个关卡** | 从"新手入门"到"终极考验"，难度逐步递增 |
| ⭐ **星星评价** | 每关可获得 1-3 星，挑战完美通关 |
| 🏆 **成就系统** | 9个成就等你解锁（五连击、传奇厨师、通关达人等） |
| 🔥 **连击系统** | 连续完成订单可获得连击加分 |
| 🏅 **排行榜** | 本地存储前10名记录 |
| 🌐 **双语支持** | 中文 / English 自由切换 |
| 🎵 **音效系统** | Web Audio API 实现的多种音效 |
| ⏸️ **暂停功能** | 随时暂停/继续游戏 |
| 💡 **提示系统** | 每关3次提示机会，帮你找到需要的食材 |
| ⚙️ **难度设置** | 简单/普通/困难 三种难度可选 |
| 📱 **响应式设计** | 完美适配手机、平板、桌面端 |

## 🚀 快速开始

### 在线游玩
直接打开 `index.html` 文件即可开始游戏，无需安装任何依赖！

### 本地运行
```bash
# 方法1：直接双击打开 index.html
# 方法2：使用 Python 启动本地服务器
python -m http.server 8000
# 方法3：使用 Node.js 的 serve
npx serve
# 方法4：使用 VS Code Live Server 插件
```

## 📖 游戏玩法

1. 🏠 在首页选择关卡、查看教程或查看成就
2. 🎯 从10个关卡中选择已解锁的关卡开始
3. 👀 观察顾客想要点什么菜
4. 👆 点击高亮提示的食材添加到锅中
5. 🍲 食材收集完成后点击"开始烹饪"
6. ⚡ 连续完成可获得连击加分！
7. 🎯 在时间限制内达到目标分数通关
8. ⭐ 根据得分获得1-3星评价
9. 🔓 通关后解锁下一关
10. 🏆 收集所有成就并在成就展示页面查看！
